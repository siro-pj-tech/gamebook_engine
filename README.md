# AI GameBook Engine v1.0

Google Gemini API (v2.5 Flash) を活用した、ブラウザだけで動作するゲームブックエンジンです。
JSONファイルを読み込むことで、ホラー、ファンタジー、SFなど様々なジャンルのテキストアドベンチャーゲームを遊ぶことができます。

## 特徴
* **AIによる自動描写**: 選択肢を選ぶと、次のシーンまでの展開をGeminiが小説風に自動生成します。
* **カセットシステム**: シナリオデータ（JSON）を入れ替えるだけで全く別のゲームになります。
* **セーブ＆ロード**: 進行状況をJSONファイルとしてダウンロードし、いつでも続きから遊べます。
* **1回限りの「戻る」機能**: 選択を間違えたとき、一度だけ時を戻せます。
* **API使用量モニター**: 無料枠を超えないよう、RPM（分間リクエスト数）などをリアルタイム表示します。

## 遊び方
1.  **準備**: Google AI StudioでAPIキーを取得してください（無料）。
2.  **起動**: [GitHub PagesのURL] にアクセスします。
3.  **入力**: APIキーを入力し、「冒険の書（JSONファイル）」を選択します。
4.  **開始**: 「物語を始める」をクリックしてスタート！

## 動作環境
* PC / タブレット / スマートフォン (モダンブラウザ)
* Google Gemini API Key が必要です
